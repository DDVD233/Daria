# Daria

<p align="center">
  <img src="assets/icons/daria.png" alt="Daria" width="128">
</p>

Daria is a client app for [Misskey](https://github.com/misskey-dev/misskey), a
federated social platform.

It is a fork of [Aria](https://github.com/poppingmoon/aria), which is itself a
fork of [Miria](https://github.com/shiosyakeyakini-info/miria). The main focus of
this fork is deeper optimization for [dvd.chat](https://dvd.chat), together with a
number of complementary server-side changes.
